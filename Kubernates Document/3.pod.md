## ðŸ“„ Option 2: Create Pod using YAML manifest

Create a file called `pod.yaml` with this content:

```yaml
apiVersion: v1
kind: Pod
metadata:
  name: my-pod
  namespace: my-namespace   # Optional: use if you created a custom namespace
  labels:
    app: nginx
spec:
  containers:
    - name: nginx
      image: nginx:latest
      ports:
        - containerPort: 80
```

Then apply it:

```bash
kubectl apply -f pod.yaml
```

Check pod status:

```bash
kubectl get pods -n my-namespace
```
