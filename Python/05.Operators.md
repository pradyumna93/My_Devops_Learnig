# âœ… **1. Arithmetic Operators**

Used for basic mathematical calculations.

| Operator | Description         | Example   | Output |
| -------- | ------------------- | --------- | ------ |
| `+`      | Addition            | `10 + 5`  | 15     |
| `-`      | Subtraction         | `10 - 5`  | 5      |
| `*`      | Multiplication      | `10 * 5`  | 50     |
| `/`      | Division            | `10 / 4`  | 2.5    |
| `//`     | Floor division      | `10 // 4` | 2      |
| `%`      | Modulus (remainder) | `10 % 4`  | 2      |
| `**`     | Power               | `2 ** 3`  | 8      |

### Example:

```python
a = 10
b = 3

print(a + b)   # 13
print(a / b)   # 3.333
print(a // b)  # 3
print(a % b)   # 1
print(a ** b)  # 1000
```

# âœ… **2. Assignment Operators**

Used to assign values to variables, especially updating values.

| Operator | Meaning               | Example   |
| -------- | --------------------- | --------- |
| `=`      | Assign                | `x = 10`  |
| `+=`     | Add & assign          | `x += 5`  |
| `-=`     | Subtract & assign     | `x -= 5`  |
| `*=`     | Multiply & assign     | `x *= 2`  |
| `/=`     | Divide & assign       | `x /= 2`  |
| `//=`    | Floor divide & assign | `x //= 2` |
| `%=`     | Modulus & assign      | `x %= 2`  |
| `**=`    | Power & assign        | `x **= 2` |

### Example:

```python
x = 10
x += 5     # now x = 15
x *= 2     # now x = 30
```

# âœ… **3. Comparison Operators**

Used to compare values (result is True or False).

| Operator | Meaning          | Example  | Output |
| -------- | ---------------- | -------- | ------ |
| `==`     | Equal            | `5 == 5` | True   |
| `!=`     | Not equal        | `5 != 3` | True   |
| `>`      | Greater than     | `5 > 3`  | True   |
| `<`      | Less than        | `5 < 3`  | False  |
| `>=`     | Greater or equal | `5 >= 5` | True   |
| `<=`     | Less or equal    | `3 <= 5` | True   |

### Example:

```python
a = 10
b = 20

print(a == b)  # False
print(a < b)   # True
print(a >= 10) # True
```

---

# âœ… **4. Logical Operators**

Used to combine conditional statements.

| Operator | Meaning                   | Example          | Output |
| -------- | ------------------------- | ---------------- | ------ |
| `and`    | True if both are true     | `True and False` | False  |
| `or`     | True if at least one true | `True or False`  | True   |
| `not`    | Reverses truth value      | `not True`       | False  |

### Example:

```python
age = 25
salary = 30000

print(age > 18 and salary > 20000)   # True
print(age > 30 or salary > 20000)    # True
print(not(age > 30))                 # True
```

---

# âœ… **5. Bitwise Operators**

Used for operations on binary numbers.

| Operator | Name        | Meaning                                  |                                    |
| -------- | ----------- | ---------------------------------------- | ---------------------------------- |
| `&`      | AND         | sets each bit to 1 if both bits are 1    |                                    |
| `        | `           | OR                                       | sets each bit to 1 if one bit is 1 |
| `^`      | XOR         | sets each bit to 1 if bits are different |                                    |
| `~`      | NOT         | flips bits                               |                                    |
| `<<`     | Left shift  | shift bits left                          |                                    |
| `>>`     | Right shift | shift bits right                         |                                    |

### Example:

```python
a = 5    # 0101
b = 3    # 0011

print(a & b)   # 1
print(a | b)   # 7
print(a ^ b)   # 6
print(a << 1)  # 10
print(a >> 1)  # 2
```

---

# âœ… **6. Membership Operators**

Used to check if a value is in a sequence (string, list, tuple).

| Operator | Meaning        | Example              | Output |
| -------- | -------------- | -------------------- | ------ |
| `in`     | exists         | `"a" in "apple"`     | True   |
| `not in` | does not exist | `"z" not in "apple"` | True   |

### Example:

```python
fruits = ["apple", "banana", "mango"]

print("apple" in fruits)       # True
print("orange" not in fruits)  # True
```

---

# âœ… **7. Identity Operators**

Check if two variables refer to the same memory location.

| Operator | Meaning           |
| -------- | ----------------- |
| `is`     | same object       |
| `is not` | different objects |

### Example:

```python
a = [1,2,3]
b = a
c = [1,2,3]

print(a is b)     # True (same memory)
print(a is c)     # False (same value, different memory)
print(a == c)     # True (values are same)
```
