### **6. Create a new service using the given attributes**

#### **Attributes:**

* **Name:** webapp-service
* **Type:** NodePort
* **Port:** 8080
* **TargetPort:** 8080
* **NodePort:** 30080
* **Selector:**

  * **name:** simple-webapp

---

#### **Option 1: Create using command**

```bash
kubectl expose pod simple-webapp \
  --name=webapp-service \
  --type=NodePort \
  --port=8080 \
  --target-port=8080 \
  --node-port=30080
```

âœ… **Explanation:**
The `kubectl expose` command creates a service directly from an existing pod or deployment.

---

#### **Option 2: Create using YAML file**

**File: `webapp-service.yaml`**

```yaml
apiVersion: v1
kind: Service
metadata:
  name: webapp-service
spec:
  type: NodePort
  selector:
    name: simple-webapp
  ports:
    - port: 8080
      targetPort: 8080
      nodePort: 30080
```

**Create the service:**

```bash
kubectl apply -f webapp-service.yaml
```

**Verify creation:**

```bash
kubectl get svc webapp-service
```
